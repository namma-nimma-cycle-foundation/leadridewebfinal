# DyteMeetingHandler

## Properties

| Property name | Data type                                 | Use                                                                                                                                                                                 |
| ------------- | ----------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| participants  | List <[DyteParticipant](./participant)>   | List of participants, including the current participant (see [participant controls and actions](./../advanced-usage#controlling-individual-participants-and-actions) for more info) |
| self          | [DyteSelfParticipant](./self-participant) | The current participant (see [self participant controls](./../advanced-usage#check-if-the-participant-is-current-participant) for more info)                                        |

## Methods

| Method name    | Params (with data type)                                                                                                                                                                     | Use                                                                                                                                                                                        |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| updateUIConfig | uiConfigOptions: [Map <String, dynamic>](./dyte-ui-config)                                                                                                                                  | Change the UI and layout of the current meeting according to the relevant params specified in the object (see [customize the meeting UI](./../customize-meeting-ui) for available options) |
| events.on      | eventName: [String](./dyte-meeting-events), context: Object?, callback: function(event: [Event](https://pub.dev/documentation/eventify/latest/eventify/Event-class.html), context: Object?) | Sets the handler for given `eventName` to `callback` function (see [events](./../events) for more details)                                                                                 |
