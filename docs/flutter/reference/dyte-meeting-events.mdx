# Events

| Event               | Triggered when                                                                                                                                                                                 |
| ------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| meetingConnected    | the participant has connected to the meeting but hasnâ€™t started producing or consuming content streams - think of this as establishing a successful connection to the meeting and nothing else |
| meetingJoin         | the participant actually joins the meeting, and can start producing as well as consuming content streams                                                                                       |
| meetingDisconnected | the participant gets disconnected from the meeting for any reason                                                                                                                              |
| meetingEnd          | the meeting is ended by the host or by the organization (as an admin function)                                                                                                                 |
| participantJoin     | another participant joins the meeting ("join" here as the same context as the `meetingJoin` event)                                                                                             |
| participantLeave    | another participant disconnects from the meeting ("disconnect" here as the same context as the `disconnect` event)                                                                             |
